### 一、项目功能

**一款带壁纸的自定义日历APP**

* 其中使用特殊颜色和形状标注出特定日期（如当前日期、双休日、本月日期、非本月日期），便于区别；
* 背景图均为精心挑选，每张图的景色都代表特定的月份，经过后期的蒙版和加月份文字水印处理，背景图对日期文字的干扰显著降低；
* 支持简单的日程记录。
